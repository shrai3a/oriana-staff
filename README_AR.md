# 🎯 Oriana Staff - نظام إدارة الموظفين والحضور

نظام متكامل وحديث لإدارة الموظفين وتتبع الحضور والرواتب مع تطبيقات منفصلة للإدارة والموظفين.

- ✅ عرض الملف الشخصي والإحصائيات

## 🛠️ التكنولوجيا المستخدمة

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **Tailwind CSS 4** - تصميم الواجهات
- **TypeScript** - لغة برمجة آمنة
- **Wouter** - توجيه الصفحات
- **Recharts** - رسوم بيانية

### Backend
- **Express.js** - خادم الويب
- **tRPC** - واجهة برمجية آمنة
- **Drizzle ORM** - إدارة قاعدة البيانات
- **MySQL/MariaDB** - قاعدة البيانات

### Tools
- **Vite** - بناء التطبيق
- **Vitest** - اختبار الوحدات
- **Prettier** - تنسيق الكود

## 📋 المتطلبات

- Node.js v16 أو أحدث
- npm أو pnpm
- قاعدة بيانات MySQL/MariaDB (محلية أو سحابية)
- متصفح حديث

## 🚀 البدء السريع

### التثبيت المحلي

```bash
# 1. استخراج الملفات
cd oriana_staff

# 2. تثبيت المكتبات
pnpm install

# 3. إنشاء ملف .env
cp .env.example .env
# ثم عدّل البيانات في .env

# 4. تشغيل الخادم
pnpm dev
```

افتح `http://localhost:3000` في المتصفح.

### النشر على السيرفر

اتبع دليل [DEPLOYMENT.md](./DEPLOYMENT.md) لنشر التطبيق على Railway أو أي سيرفر آخر.

## 🎯 كيفية الاستخدام

2. اختر "دخول الإدارة"

4. ستنتقل إلى لوحة التحكم الإدارية

**الميزات المتاحة:**
- إدارة الموظفين والفروع

- حساب الرواتب
- التقارير والإحصائيات

### دخول الموظفين

1. افتح التطبيق
2. اختر "دخول الموظفين"
3. سجّل دخول بحسابك
4. ستنتقل إلى تطبيق الموظفين

**الميزات المتاحة:**
- تسجيل الحضور والانصراف
- عرض الملف الشخصي
- عرض الإحصائيات

## 🐛 استكشاف الأخطاء

### المشكلة: "Cannot find module"
**الحل:**
```bash
pnpm install
```

### المشكلة: "Database connection failed"
**الحل:**
- تحقق من رابط قاعدة البيانات في `.env`
- تأكد من أن قاعدة البيانات تعمل
- تأكد من أن كلمة المرور صحيحة

### المشكلة: "Port 3000 already in use"
**الحل:**
```env
PORT=3001
```
